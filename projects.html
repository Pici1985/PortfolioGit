Timer

A watch and stopwatch app to count study time.

This app is built with pure vanilla JS. The clock shows the actual time getting data from the built in clock. Stopwatch can be started, stopped, cleared. On submit a line appears with the lenght of the current study session. This data is saved into a JSON file. With the use of JSON-server package. Session data can be retrieved from db.JSON once everything is properly set up for the app to work. See README on GitHub.
